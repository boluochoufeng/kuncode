# KunCode

KunCode is a Rust-native coding agent harness runtime.

It implements the harness patterns described in `shareAI-lab/learn-claude-code`
as a model-agnostic, type-safe, auditable production runtime.

See `docs/specs/` for design and MVP plan:

- [kuncode-agent-harness-design.md](docs/specs/kuncode-agent-harness-design.md): long-term architecture.
- [kuncode-mvp-development-plan.md](docs/specs/kuncode-mvp-development-plan.md): MVP scope and phases.
- [kuncode-agent-harness-rationale-gaps.md](docs/specs/kuncode-agent-harness-rationale-gaps.md): open design questions.
